# FactoryAI 智工助手 - 产品设计文档 (MVP)

## 版本：v1.0 精简版

---

## 一、产品定位

**核心价值**：规范工程师每日工作，提升执行效率

**目标用户**：
- 执行层：生管/设备/品保工程师
- 管理层：主管

---

## 二、核心功能（精简为4个）

### 1. 任务列表
- 工程师看到今日分配的任务
- 按优先级显示
- 状态：待执行/进行中/已完成

### 2. 任务执行
- 每项任务有具体步骤（3-5步）
- 步骤类型：勾选确认 / 文字输入 / 拍照存档
- 进度实时显示

### 3. 任务提交
- 全部步骤完成后才能提交
- 提交后自动归档

### 4. 主管分配（简化）
- 主管分配任务给工程师
- 选择任务模板

---

## 三、任务模板示例

| 角色 | 任务类型 | 执行步骤 |
|-----|---------|---------|
| 生管 | 生产排配确认 | 1.确认排配 2.记录异常 3.提交 |
| 生管 | 产能记录 | 1.采集数据 2.录入系统 3.异常分析 |
| 设备 | 设备点检 | 1.外观检查 2.参数记录 3.异常记录 |
| 设备 | 设备保养 | 1.停机确认 2.清洁保养 3.更换配件 4.试机 |
| 品保 | IQC检验 | 1.来料检验 2.记录数据 3.判定结果 |
| 品保 | HOLD复判 | 1.确认原因 2.复判决策 3.记录归档 |

---

## 四、用户体验流程

```
主管分配任务 → 工程师收到通知 → 执行任务 → 完成提交 → 自动记录
```
